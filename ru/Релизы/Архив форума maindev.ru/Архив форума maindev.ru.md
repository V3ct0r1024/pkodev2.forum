# Архив форума maindev.ru

![](images/0.png) / ![](images/1.png)

До февраля 2016 года наш форум базировался на сайте **maindev.ru** как один из разделов, посвященных разработке MMORPG-игр. К сожалению, maindev.ru стал часто работать с перебоями, а даунтаймы достигали месяцев, поэтому было принять решение создать новый форум — PKODev.NET.

За годы работы форума maindev.ru накопилось очень много полезных материалов и знаний, поэтому выкладываем его дамп в виде набора HTML-страниц.

В архиве "**Архив форума maindev.ru**" находятся две версии сайта:
* Т.н. "старый" **maindev.ru — до 2015 года**:
  ```
  Откройте файл ".\Архив форума maindev.ru\Старый maindev.ru (до 2015 года)\Dev\index.html"
  ```

* Т.н. "новый" **maindev.ru — после 2015 года**:
  ```
  Откройте файл ".\Архив форума maindev.ru\Новый maindev.ru (после 2015 года)\index.html"
  ```
