# Допустимые e-mail домены для регистрации

Всем привет!

Из‑за участившихся в последнее время регистраций спам‑ботов и в целях борьбы со спамом на форуме был введён белый список e‑mail доменов для регистрации новых пользователей.

Зарегистрироваться на форуме можно, используя электронные ящики следующих серверов:

```
mail.ru
list.ru
bk.ru
yandex.ru
yandex.com
ya.ru
msn.com
outlook.com
gmail.com
hotmail.com
hotmail.es
yahoo.com
yahoo.co.uk
```

Регистрация новых пользователей с использованием e‑mail с другими доменами запрещена.

В данной теме принимаются предложения для включения в белый список других крупных почтовых серверов, которые по какой-либо причине не попали в белый список.
